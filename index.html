Ícones

1. Para preencher a borda do ícone com alguma cor, existem duas classes chamadas "icone-preenchimento-preto", "icone-preenchimento-ciano", "icone-preenchimento-amarelo". Use-as na tag "path" do ícone que você deseja preencher com a respectiva cor.

2. Para adicionar o fundo circular com sombra, existe uma classe chamada "icone-com-fundo", adicione esta classe na tag svg caso deseje esse comportamento.
